# Priscripto-MERN-
A medical field app helps people take appointments with top doctors with minimal effort.. 


1. Frontend 
To run the front-end, just open this folder in VS-Code and open the integrated terminal, now run the command: "npm run dev".
It will open a link to the local host and port where you can view the frontend.

Note-(In case it is not working and showing error, run once in your integrated terminal cmd: "npm install".
It will install all dependencies for the project.)

2. Backend
To run the Backend, just open this folder in VS Code and open the integrated terminal Now, run the command: "npm run server".
It will open a link to the local host where it shows "server is running at port no, and Database is connected," and the port where you can view the Backend.

Note-(In case it is not working and showing error, run once in your integrated terminal cmd: "npm install".
It will install all dependencies for the project and delete the  vite/.dev in the node module.)

3. Admin 
To run Admin, just open this folder in VS-Code and open the integrated terminal, now run cmd: "npm run dev".
It will open a link to the local host and port where you can view the Admin.

Note-(In case it is not working and showing error, run once in your integrated terminal cmd: "npm install".
It will install all dependency for the project.)
